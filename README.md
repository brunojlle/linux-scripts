# linux-scripts
  
This repository provides scripts and step-by-step guides to automate and simplify the setup of essential development tools on Linux systems. Here you will find:

- **Anaconda installation and configuration**: Automate the setup of the Anaconda Python distribution for data science and development.
- **Zsh and plugins setup**: Install and configure the Zsh shell, Oh My Zsh, and useful plugins for a more productive terminal experience.
- **GitHub SSH authentication**: Step-by-step instructions to securely connect to GitHub using SSH keys.

These resources are ideal for quickly preparing a Linux environment for programming, data analysis, and version control.
